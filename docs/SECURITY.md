# Política de Segurança (SECURITY)

## Versões Suportadas

Atualmente, apenas a versão principal (Latest) recebe atualizações de segurança.

| Versão | Suporte |
| :--- | :--- |
| **0.1.x** | ✅ Ativo |
| **< 0.1.x** | ❌ Não suportado |

## Reportando Vulnerabilidades

Estamos comprometidos em garantir a segurança dos dados logísticos e de inventário. Se você descobrir uma falha, por favor siga estes passos:

1. **Não abra uma Issue pública** para reportar vulnerabilidades de segurança.
2. Envie um e-mail detalhado para `seguranca@blacktowerx.com.br` (exemplo).
3. Inclua passos para reproduzir, impacto potencial e (se possível) sugestões de correção.
4. Responderemos em até 48 horas úteis para confirmar o recebimento e iniciar o processo de correção.

## Nossa Promessa

- Manteremos você informado sobre o progresso da correção.
- Após a correção ser lançada, daremos os devidos créditos (se desejado) em nossos registros de segurança.

Obrigado por ajudar a manter o **Inventário Ágil** seguro!
